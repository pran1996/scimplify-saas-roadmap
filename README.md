# Scimplify — Digital Product & SaaS Revenue Roadmap

Interactive MVP demos for Scimplify's digital product strategy.

**6 Products:**
1. RFQ Intelligence + China+1 Assessment
2. IONS Project Hub (R&D + Molecule Customization)
3. ChemGraph (Network Intelligence + Cost Benchmarking)
4. ComplianceOS (Regulatory & Safety SaaS)
5. ATOMS Commerce (Self-Serve E-Commerce)
6. GreenScore (ESG & Carbon Tracking)

## Run Locally
```bash
npm install
npm start
# Open http://localhost:3000
```

## Deploy
Hosted on Railway. Push to main branch to auto-deploy.
